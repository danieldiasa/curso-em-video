Principais Formatações de Texto



Negrito

Para deixar o texto em negrito, utilize a tag <strong>.


Itálico

Para deixar o texto em itálico, utilize a tag <em>.


Texto Marcado

Para marcar o texto, utilize a tag <mark>. Exemplo: Texto marcado.


Texto Riscado

Para riscar o texto, utilize a tagu <del>. Exemplo: Texto riscado.


Texto Sublinhado

Para sublinhar o texto, utilize a tag <ins>. Exemplo: Texto sublinhado.


Texto Grande e Pequeno

Para deixar o texto maior, utilize a tag <big>. Exemplo: Texto grande. Para deixar o texto menor, utilize a tag <small>. Exemplo: Texto pequeno.


Texto Sobrescrito

Para sobrescrever o texto, utilize a tag <sup>. Exemplo: E = mc2.


Texto Subscrito

Para subscrever o texto, utilize a tag <sub>. Exemplo: H2O.
