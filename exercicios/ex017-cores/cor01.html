Cores em CSS

As cores em CSS podem ser definidas de várias maneiras, incluindo nomes de cores, valores hexadecimais, RGB, RGBA, HSL e HSLA. Aqui estão alguns exemplos:

Nomes de Cores

  • red
  • blue
  • green
  • yellow
  • black
  • white

Valores Hexadecimais

  • #FF0000 (vermelho)
  • #0000FF (azul)
  • #00FF00 (verde)
  • #FFFF00 (amarelo)
  • #000000 (preto)
  • #FFFFFF (branco)

Valores RGB

  • rgb(255, 0, 0) (vermelho)
  • rgb(0, 0, 255) (azul)
  • rgb(0, 255, 0) (verde)
  • rgb(255, 255, 0) (amarelo)
  • rgb(0, 0, 0) (preto)
  • rgb(255, 255, 255) (branco)

Valores RGBA

  • rgba(255, 0, 0, 1) (vermelho opaco)
  • rgba(0, 0, 255, 0.5) (azul semi-transparente)
  • rgba(0, 255, 0, 0.3) (verde semi-transparente)

Valores HSL

  • hsl(0, 100%, 50%) (vermelho)
  • hsl(240, 100%, 50%) (azul)
  • hsl(120, 100%, 50%) (verde)

Valores HSLA

  • hsla(0, 100%, 50%, 1) (vermelho opaco)
  • hsla(240, 100%, 50%, 0.5) (azul semi-transparente)
  • hsla(120, 100%, 50%, 0.3) (verde semi-transparente)
