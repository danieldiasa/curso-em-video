Código-Fonte

Para exibir código-fonte em HTML, você pode usar as tags <code> e <pre>.


Tag code

A tag <code> é usada para marcar trechos de código dentro de um parágrafo e para obter o mesmo espaçamento em todos os caracteres.

Tag pre

A tag <pre>é usada para preservar a formatação do texto, incluindo espaços e quebras de linha. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Exemplo de Código</title>
  </head>
  <body>
    <h1>Exemplo de Código</h1>
    <p>Este é um exemplo de como usar as tags <code> e <pre>.</p>
    <pre>
      <code>console.log('Olá, mundo!');</code>
    </pre>  
  </body>
</html>

Comentários

Para adicionar comentários em HTML, você pode usar a sintaxe <!-- comentário -->.

Comentários são úteis para deixar notas ou explicações no código que não serão exibidas na página web.

Exemplo de comentário em HTML: 

<!-- Este é um comentário em HTML -->


Abreviações

Para adicionar abreviações em HTML, você pode usar a tag <abbr>.

A tag <abbr> é usada para definir abreviações ou siglas, e você pode fornecer uma descrição completa usando o atributo title.

Exemplo de abreviação em HTML: HTML é a linguagem de marcação usada para criar páginas web.


Texto Invertido

Para inverter o texto em HTML, você pode usar a tag <bdo> com o atributo dir.

A tag <bdo> (bidirectional override) é usada para controlar a direção do texto. Você pode definir o atributo dir="rtl" para inverter o texto da direita para a esquerda.
